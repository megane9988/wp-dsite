Display the total number of Hatena Bookmarks, Facebook Likes and Tweets of your post/page.

== Description ==

BLT Counter displays the total number of Hatena Bookmarks, Facebook Likes and Tweets of your post/page, using a custom template tag anywhere inside the Loop in your theme template.

== Installation ==

1. Upload the entire `blt-counter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the tag <?php blt_counter_show(get_permalink()); ?> where you like.
